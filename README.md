# summary

A text summarization jar.

## Installation

Download from https://github.com/recursiveMake/summary

## Usage

Run with:

    $ java -jar summary-0.1.0-standalone.jar [OPTIONS]

## Options

`-f FILE` File to summarize
`-l LIMIT` Lines to limit text
